# VidBRZ
a simple, lightweight web based video search tool for tmdb and provides links to vidsrc


# This tool is not intended for piracy, please do not misuse this small project for content that you do not own rights to

## Requiremets
Python must be installed on your machine and the following packages must be installed:

```
tmdb_api
```

## Installation:

Save the latest release zip file, extract it, and run "flask_server.py"

Web app can be accesssed at "http://localhost:8000" on the same machine that the server is running on.
